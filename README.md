## OS dependencies
+ Tensorflow-gpu==2.3.2
+ Python==3.9.0
+ keras==2.3.0
+ hyperopt==0.2.5
+ pandas=1.4.4
+ numpy=1.21.5


The model is included in anothertry.py and data augmentation methods is writed in dataag.py
